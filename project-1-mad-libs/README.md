# Project 1 - Mad Libs

Teacher: David Steedman

This repository contains Project 1 and the corresponding labs for the MHS Software Engineering class. 
Each lab focuses on developing specific beginner programming skills in Python.

### Lab 1: Hello World
- File: [`lab1.py`](lab1.py)
- Description: Write a program that prints out "Hello, World!".

### Lab 2: Hello, name
- File: [`lab2.py`](lab2.py)
- Description: Write a program that asks for and greets a user by name.

### Lab 3: Hello, full name
- File: [`lab3.py`](lab3.py)
- Description: Ask for first and last name and greet user by their full name.

### Lab 4: Split String
- File: [`lab4.py`](lab4.py)
- Description: Ask for full name and greet user by their first name only.

### Mad Libs Game
- File: [`mad-libs.py`](mad-libs.py)
- Description: Complete a functional Mad Libs game in Python.

## Expected Output

### Lab 1 - Hello World
```
Hello, World!
```

### Lab 2 - Hello, name
```
What is your name? Alice
Hello, Alice!
```

### Lab 3 - Hello, full name
```
What is your first name? Alice
What is your last name? Smith
Hello, Alice Smith!
```

### Lab 4 - Split String
```
What is your full name? Alice Smith
Hello, Alice!
```

### Mad Libs Game (Example)
```
Welcome to Mad Libs!
Enter an adjective: silly
Enter a noun: elephant
Enter a verb: dancing
Enter an adverb: quickly

Here's your story:
The silly elephant was dancing quickly through the forest.
```
