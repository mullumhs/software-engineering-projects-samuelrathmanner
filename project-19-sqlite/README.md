# Flask SQLite Project

Teacher: David Steedman

This repository contains four tasks for the MHS Software Engineering class focusing on Flask and SQLite. This project introduces students to working with databases in web applications using Flask and SQLite.

## SQLite Tasks

### Task 1: SQL Basics

- File: N/A (Uses online SQL playground)
- Description: Introduction to SQL syntax and basic database operations.
- This task covers:
  - Creating tables
  - Inserting, selecting, updating, and deleting data
  - Basic and advanced queries

### Task 2: SQLite Basics

- File: `app.py`
- Description: Set up SQLite in Python and perform basic database operations.
- This task covers:
  - Connecting to SQLite databases
  - Creating tables
  - Inserting, querying, updating, and deleting data
  - Implementing a simple movie database manager

### Task 3: SQLite and Flask

- Files: `app.py`, `templates/*.html`
- Description: Integrate SQLite with Flask to create a web-based movie database.
- This task covers:
  - Setting up Flask with SQLite
  - Creating forms to add movies
  - Displaying movies in HTML tables
  - Implementing search functionality

### Task 4: Object-Relational Mapping (ORM)

- Files: `app.py`, `templates/*.html`
- Description: Introduce ORM concepts using Flask-SQLAlchemy.
- This task covers:
  - Setting up Flask-SQLAlchemy
  - Defining models and creating tables
  - CRUD operations using ORM
  - Implementing search and edit features

## How to Submit Your Work

1. Accept the assignment and clone this repository to your local machine.
2. Create the necessary Python and HTML files for each task.
3. Implement the required functionality for each task.
4. Commit your changes with meaningful commit messages.
5. Push your changes to your GitHub repository.
